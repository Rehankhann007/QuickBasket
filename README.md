
# 🛒 QuickBasket

A modern Full Stack MERN Grocery Delivery Application with Authentication, OTP Verification, Google Login, Admin Dashboard, Delivery Tracking and Real-Time Order Updates.

## 🚀 Live Demo

🌐 Frontend:
https://quick-basket-xi.vercel.app/

⚙️ Backend API:
https://quickbasket-2zvt.onrender.com

---

## ✨ Features

### 👤 Authentication

- Email & Password Login
- Google Login
- JWT Authentication
- OTP Email Verification
- Forgot Password
- Protected Routes
- Role Based Authentication

### 🛍 User

- Browse Products
- Add to Cart
- Update Quantity
- Checkout
- Place Orders
- Track Orders
- Live Order Status

### 👨‍💼 Admin

- Dashboard
- Product Management
- Order Management
- User Management
- Confirm Orders
- Assign Delivery Boy

### 🚚 Delivery Boy

- Assigned Orders
- Live Delivery Tracking
- OTP Verification
- Delivery Status Update

---

## 🛠 Tech Stack

Frontend

- React.js
- Vite
- Tailwind CSS
- Axios
- React Router
- Framer Motion
- Socket.io Client

Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Nodemailer
- Socket.io

Deployment

- Vercel
- Render
- MongoDB Atlas

## ⚙️ Installation

Clone Repository

```bash
git clone https://github.com/Rehankhann007/QuickBasket.git
```

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Backend

```
PORT=
MONGO_URI=
JWT_SECRET=
EMAIL_USER=
EMAIL_PASS=
GOOGLE_CLIENT_ID=
CLIENT_URL=
```

Frontend

```
VITE_API_URL=
VITE_GOOGLE_CLIENT_ID=
```
